# Data Cleaning & Reporting Automation

## 📌 Project Overview
This project focuses on cleaning raw employee data and creating an interactive Power BI dashboard for automated reporting. The dataset was cleaned by handling missing values, removing duplicate records, and ensuring consistent data types before visualization.

## 🎯 Objective
- Clean and preprocess raw data.
- Handle missing values and duplicate records.
- Create an interactive Power BI dashboard.
- Generate automated reports for better decision-making.

## 🛠️ Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- Power Query
- DAX

## 📂 Project Structure
```
Data_Cleaning_Automation/
│── employee_data.xlsx
│── Data_Cleaning_Report.pbix
│── README.md
```

## 📊 Data Cleaning Process
- Imported Excel data into Power BI.
- Removed duplicate records.
- Handled missing values.
- Corrected data types.
- Loaded cleaned data into the Power BI model.

## 📈 Dashboard Features
- Total Employees (Card)
- Total Salary (Card)
- Average Salary (Card)
- Employees by Department (Pie Chart)
- Salary by Department (Column Chart)
- Employee Details (Table)

## 📋 Key Insights
- Improved data quality through preprocessing.
- Automated reporting with interactive visualizations.
- Enhanced decision-making using clean and reliable data.

## 🚀 Outcome
This project demonstrates the importance of data cleaning and reporting automation using Power BI. It highlights how preprocessing and visualization can improve business reporting efficiency.
